# 03 Normalize Evidence

输出 `03_normalized_evidence_pack.md`。把来源不一、格式不一的材料整理成可比、可追溯的证据包。

必要时先读 `../references/evidence-rules.md`。

## 目标

- 统一字段
- 去重和归类
- 标注来源可信度
- 区分证据、推断、假设和缺口

## 输出结构

至少包含：

```text
Market Evidence Schema
Product Evidence Schema
User Voice Evidence Schema
Competitor Evidence Schema
Source List
Confidence Notes
Assumptions
Evidence Gaps
```

## 规则

- 保留来源链路，确保后续结论可追溯。
- 对弱证据单独标注，不要混入强证据。
- 同一事实如来自多源，合并时保留来源列表。
