# 05 Build User Model

输出 `05_user_model.md`。把用户评论、访谈、论坛讨论转成用户分群、场景和需求结构。

## 目标

- 识别用户分群
- 归纳典型使用场景
- 提炼痛点与隐性需求
- 连接到产品维度

## 输出结构

至少包含：

```text
User Segments
Usage Scenarios
Raw Voice Structuring
Pain Points
Implied Needs
Product Dimensions
Representative Evidence
```

## 规则

- 保留代表性原话或证据摘要。
- 不要把个体抱怨直接写成普遍结论。
- 明确哪些判断只是趋势，哪些已经有足够样本支持。
