## Product Attribute Taxonomy

1. 补能能力
2. 续航确定性
3. 高速舒适性
4. 车内休息能力
5. 装载与取物效率
6. 智驾减负能力
7. 家庭协同能力
8. 旅途服务保障

## Spec-to-Experience Mapping

| Spec / Capability | Not Equal To | Mapped Experience |
|---|---|---|
| CLTC 长续航 | 不直接等于“长途无焦虑” | 需要叠加补能速度、排队概率、天气影响、满载衰减 |
| 5C 超充 / 超充网络 | 不直接等于“省心” | 只有当站点可达、可用、可预判排队时，才转化为低决策负担 |
| 增程/插混综合续航 1300km+ | 不直接等于“体验更好” | 它解决的是规划成本，但不自动解决 NVH、底盘、休息舒适 |
| 大后备箱 / 前备箱 | 不直接等于“家庭好用” | 取物路径、分层收纳、湿干分离、儿童物品快速拿取更关键 |
| 高阶智驾 | 不直接等于“疲劳更低” | 需要稳定可预期、对大车安全感高、接管时机清晰 |
| 按摩/零重力/后排调节 | 不直接等于“长途舒适” | 要看是否支持全家多角色轮换休息、儿童安全座椅兼容、噪音抑制 |

## Competitor Positioning

- Model Y：高效率纯电路线
- 理想 L6：家庭五座增程路线
- 小鹏 G9：快充纯电舒适路线
- 智界 R7：智能生态大空间路线
- 问界 M7：六座灵活 + 高阶智驾路线

## Competitor Strengths

- 纯电阵营把“补能时间”压得越来越短
- 增程阵营把“长途规划成本”降得越来越低
- 主流车型都能给出大空间、舒适配置、辅助驾驶卖点

## Competitor Weaknesses

- 多数产品仍把“长途体验”拆成参数卖点，而非完整旅程
- 对家庭旅途中“谁累、谁拿东西、谁照顾孩子、何时停、停哪儿”支持不足
- 智驾减负与用户信任之间仍有明显缝隙

## Attackable Gaps

1. 旅程规划与补能决策仍然是用户脑力负担
2. 长途舒适聚焦前排驾驶者，未系统服务全家
3. 车内收纳和在途生活支持碎片化
4. 智驾从“能开”到“让家庭放心用”仍未跨越

## Learnable Practices

- 向纯电学习：补能效率、补能可视化、路线联动
- 向增程学习：把规划成本降到最低
- 向六座学习：角色分区和照顾老人/孩子的便利性
