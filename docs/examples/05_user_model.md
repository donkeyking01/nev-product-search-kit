## User Segments

### Segment A：三口/四口城市家庭

- 主要场景：周末短途 + 节假日高速自驾
- 典型诉求：省心、省时、不折腾

### Segment B：二孩/多代同行家庭

- 主要场景：中长途探亲、旅游
- 典型诉求：座位灵活、上下车方便、老人孩子不累

### Segment C：科技接受度高的年轻家庭

- 主要场景：愿意用智驾、车机和生态服务换减负
- 典型诉求：能托管一部分任务，但要可控可信

## Usage Scenarios

1. 周末 200-400 公里周边自驾
2. 节假日 500-1000 公里跨城/返乡
3. 满载 4-5 人 + 婴儿车/行李/零食/露营物品
4. 高速行驶 3-6 小时，中途需要休息、补能、喂养、换乘员

## Raw Voice Structuring

- `Evidence` 纯电用户认可快充和网络，但仍会提“跑长途要规划”“冬季/高速更焦虑”（S10/S12）
- `Evidence` 增程用户认可“说走就走”“没有里程焦虑”（S11/S12）
- `Evidence` 用户反复提到舒适、隔音、底盘、后排空间、装载便利、智驾疲劳减轻（S10/S11/S12）
- `Evidence` 官方持续强化按摩、零重力、后排调节、空气悬架、健康座舱（S4/S5/S7）

## Pain Points

1. 补能不一定缺设施，但仍缺“最优下一步”的确定性
2. 高速长时间驾驶时，全家累点不同，产品往往只顾驾驶员
3. 行李、食品、儿童用品、脏湿物品缺少真正顺手的管理方式
4. 智驾能减负，但用户不敢完全放松
5. 五座舒适与六座灵活之间仍有明显取舍成本

## Implied Needs

1. 把长途决策从“人脑临场处理”变成“系统提前编排”
2. 从“座椅舒适”升级为“全家轮换休息系统”
3. 从“空间够大”升级为“东西拿得到、放得对、途中不乱”
4. 从“智驾能力强”升级为“家人坐着放心”

## Product Dimensions

- 补能与导航协同
- 驾驶与乘坐疲劳管理
- 儿童/老人友好设计
- 模块化收纳与休息模式
- 可信赖的辅助驾驶

## Representative Evidence

- Model Y 车主：快充好，但长途开久了会累（S10）
- 理想 L6 车主：增程长途省心，但底盘晃动、局部功能覆盖不足（S11）
- 智界 R7 / 小鹏 G9 用户：高速智驾确实减负，但胎噪、靠右、贴大车等问题影响安全感（S12）
