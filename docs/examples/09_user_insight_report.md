## User Insight Report

### Insight 1

家庭用户最怕的不是没有续航，而是不确定下一步该怎么做。

- `Evidence` 高速服务区覆盖率已经很高，但节假日充电量持续创新高（S9）
- `Evidence` 纯电用户仍提规划与长途焦虑（S10/S12）
- `Inference` 产品机会在“把决策前移”，而不是继续单独卷续航数字

### Insight 2

“长途舒适”不是前排座椅舒适，而是全家都不累。

- `Evidence` 用户反馈聚焦到底盘、隔音、疲劳、后排舒适、带娃体验（S10/S11/S12）
- `Evidence` 官方大量投入零重力、按摩、后排调节、健康座舱（S4/S5/S7）
- `Inference` 说明需求真实存在，但目前仍靠零散配置拼凑

### Insight 3

空间够大，不代表旅途中好用。

- `Evidence` 官方都在强调储物/后备箱容积（S2/S4/S5）
- `Evidence` 用户真实需求是纸巾、奶瓶、零食、平板、药品、垃圾袋、湿衣物这些高频小物的拿取效率（由口碑与场景推导）
- `Inference` 家庭长途更需要“组织能力”，而不是单纯容积

### Insight 4

用户欢迎智驾减负，但拒绝把家人的安全感交给不稳定的风格。

- `Evidence` 多品牌官方都强调辅助驾驶限制（S5/S7）
- `Evidence` 用户反馈出现“贴大车”“靠右”“仍需盯防”等问题（S12）
- `Inference` 机会点是“可解释、可预期、偏保守”的家庭模式
